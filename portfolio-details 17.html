In this project I used the pandas library in python to take a dataset and clean it as much as possible. From the beginning, we imported the pandas library to read the excel document and used many functions to remove duplicates, drop columns, replace characters, use the lambda function to create a set format, and use dictionary key value notation to assign new columns.
